The most simple css framwork

What is draft.css

Framwork is targeted at quickly creating prototypes of forms and simple template. Designed to written html code looked beautiful and did not require any class and non-standard elements.

How use

All you have to do is add this code to the head section:

<link rel="stylesheet" href="https://kozakbartosz.pl/draft/def.css" />
